Who, What, Why & How?

Who: The Creator

Hi my name is Brad Jones and I built these packages.

I am a passionate ICT Professional, with a flair for Web Design & Development!
Okay granted my Design skills need some improvement...

What: This is not a Framework!

I repeat again this is not an MVC framework - like Laravel, CakePHP, Kohana, etc.

These packages are simply just that a collection of packages.

Most of the packages are standalone, they do one thing and they do it with awesomeness!

Why: I Need A Good Set of Tools

Just like a tradesman needs his tools, as a developer I also need a set of tools to help me do my job.

All of these packages have started life in a real world project.

When I find my self searching through an old project for some code it's usually time to create a new package.

How: Through the Power of Composer

In all seriousness I sometimes forget how to code in PHP without composer.

It makes life so easy, I really don't know how we did it before...

So a big thanks to Jordi Boggiano
